A MySQL-based E-Commerce Database Management System that manages customers, products, and orders using relational database concepts, primary/foreign keys, and SQL queries for sales analysis.
# E-Commerce Database Management System

## Project Overview
This project implements an E-Commerce Database Management System using MySQL. It manages customer information, product details, and order transactions through a relational database design.

## Features
- Customer Management
- Product Management
- Order Tracking
- Primary and Foreign Key Relationships
- Sales Analysis Queries

## Database Schema

### Customers
- CustomerID (Primary Key)
- CustomerName
- City

### Products
- ProductID (Primary Key)
- ProductName
- Category
- Price

### Orders
- OrderID (Primary Key)
- CustomerID (Foreign Key)
- ProductID (Foreign Key)
- Quantity
- OrderDate

## Technologies Used
- MySQL
- SQL

## Sample Queries
- Retrieve customer orders
- Calculate total sales
- Find top-selling products
- Analyze customer purchase history

## Learning Outcomes
- Database Design
- SQL Queries
- Joins
- Aggregate Functions
- Data Integrity using Primary and Foreign Keys
